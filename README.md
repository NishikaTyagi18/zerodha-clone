# 🚀 Zerodha Clone - Frontend React App

A fully responsive and interactive frontend clone of the popular stock trading platform **Zerodha (Kite)**. Built using modern web technologies to simulate a real-world financial dashboard.

## ✨ Features
* 📊 **Interactive Dashboard:** Clean interface to track market trends and stock lists.
* 📱 **Fully Responsive:** Optimised for desktop, tablets, and mobile views.
* 🎨 **Modern UI:** Built with sleek components replicating the real Kite platform experience.

## 🛠️ Tech Stack
* **Frontend:** React.js, JavaScript (ES6)
* **Styling:** Bootstrap, CSS3, HTML5
* **Deployment:** Vercel
